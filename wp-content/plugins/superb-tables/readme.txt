Create responsive tables easily & insert them with a shortcode. Lots of different designs, functions and light weight code.

== Description ==

Create simple responsive tables easily, the code output is light weight and pure HTML. When you have inserted your table contents you get a shortcode you can insert in a widget, post, page or anywhere else on your website you want to create a table.


== Features ==

- Table Shortcodes
- Multiple Color Schemes
- Use Themes Own Table Design
- Create Unlimited Tables
- Create Unlimited Rows
- Create unlimited Columns
- Purple Color Scheme
- Black/White Color Scheme
- Swap around on columns & rows with drag n' drop
- Delete Tables & Remove Them From Your Database
- Friendly User Support :-)


You can use the tables you generate to create a comparison table and display them easily in Gutenberg through the shortcodes block. 

If you have an affiliate website or  eCommerce shop built with WooCommerce, then its often necessesary to have a pricing table to compare price plans or products. You can create such tables fast and easy with our tables. The tables are responsive and schema mark up can be toggled on or off for SEO optimization. All the tables created through this WordPress plugin are shown on your website with CSS and HTML.


== Guide to creating your Superb Tables ==

1. In the wordpress admin panel, go to the sidebar and click 'Superb Tables'.
2. Click the green 'Add new table' in the top left corner
3. Fill in your table data, click 'Save'
4. Click 'Superb Tables' in the left sidebar and copy paste the 'Table Shortcode'
5. Insert the Table Shortcode (For example [spbtbl_sc id=1] ) in a post, page, widget or in HTML.

You're done!
